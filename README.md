# Portfolio Site

A minimalist, document-style portfolio built with React and Vite.

## Features
- React + Vite + React Router
- Pure CSS design system
- Accessible and responsive
- Netlify form integration

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
